# ∫ Mathe-Portal — Mathematik, die universelle Sprache

> f'(x) = 2x ist in Lagos genauso wahr wie in Hamburg.
> Mathematik gehört allen — überall, in jeder Sprache.

Interaktives Mathe-Portal mit gamifizierten Übungen.
Open Educational Resources — frei nutzbar, anpassbar, teilbar.

## Module

- **Quadratische Funktionen** — 6 Level, 24 Übungen
- **Differentialrechnung** — 4 Level, 16 Übungen (weitere geplant)

## Demo

[lernmodule.dirk-schulenburg.net/mathe-portal/](https://lernmodule.dirk-schulenburg.net/mathe-portal/)

## Quick Start

```bash
npm install
npm run dev
```
